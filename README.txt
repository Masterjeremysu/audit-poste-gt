
Application Métier GT - Version Complète

Étapes pour lancer le projet :
1. Ouvrir un terminal dans ce dossier.
2. Lancer : npm install
3. Puis : npm run dev
4. Ouvrir le navigateur sur http://localhost:5173

Dossiers inclus :
- routes : Login, Dashboard, Audits
- components : Navbar
- CSS : Tailwind prêt à l'emploi

/!\ Assure-toi que Node.js est installé
/!\ Pour le logo GT Logistics, place un fichier nommé gt-logistics-logo.png dans src/assets/
